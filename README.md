# JRX-Fitness
A MATLAB app that analyzes wearable sensor and heart rate data to evaluate workout efficiency and provide personalized fitness recommendations.

Mini-Challenge: Biomedical (Human Performance): Use data from wearable sensors and heart rate monitors to automatically identify different exercises and calculate how efficient a workout really is.

# Overview
Users aren’t effectively using their time at the gym to meet their fitness goals. So our app will allow people to discover the best way to optimize their work out while still prioritizing the limits of their strength and health. The app helps users understand whether their workouts are aligned with their fitness goals by analyzing heart rate, workout duration, and activity type. It also tracks workout history to monitor progress over time, while comparing it to average athletic data. 

# Data Processing
The app imports wearable sensor data from MATLAB `.mat` files and uses MATLAB functions 

# Features
1. Select your fitness goal, age range, and fitness level.
2. Enter workout information (start time, end time, maximum heart rate, and activity).
3. The app processes the data:
  - Analyze heart rate and workout data
  - Calculate workout efficiency
  - Personalized workout feedback along with future recommendations  
4. Review previous workouts from the Workout History page.

# Installation
1. Download the .mlapp file from the repository above
2. Search MATLAB Online in your browser and open 'MATLAB Online - MATLAB & Simulink'
3. Drag the .mlapp file into the left-hand side of the files panel 
4. Where it says “Command Window”, type ‘appdesigner’ and press enter 
5. This will prompt the App Designer Window where you can select the .mlapp file you would like to upload 
6. Once that is uploaded, select the run button where the application can then be in use!
